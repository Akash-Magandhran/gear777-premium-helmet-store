## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/gear777.git
```

Navigate to project

```bash
cd gear777
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
